1. На основании примера из книги по работе с дейтограмами сделать задание,
которое создаст соединение между 2-мя компьютерами для обмена информацией.
КОММЕНТАРИИ К ЗАДАЧЕ.
Программа дает три вариант действий:
        0-Указать IP-адрес
        1-Использовать IP-адрес по умолчанию (указан мой IP)
        2-Использовать IP-адрес локального компьютера (127.0.0.1)
==================================================================================
2. Создать текстовый чат, в котором будет серверная часть, где будут
регистрироваться клиенты и клиентская часть, с помощью которой будут
отправляться сообщения и в нее сервер будет присылать сообщения,
адресованные зарегистрированному клиенту.
КОММЕНТАРИИ К ЗАДАЧЕ.
0. Тестировать программу надо в консоли IntelliJ IDEA. Cтандартная
консоль почему-то ведет себя по-другому.
1. Чтобы запустить сервер, надо по очереди выбрать в меню следующие пункты:
        2 - CHAT
        1 - Start the server
2. Сервер выполнен как синглтон, поэтому в случае случайного перезапуска сервера,
информация о текущих соединениях потеряна не будет.
3. Чтобы зайти в чат, надо выбрать в меню такие пункты:
        2 - CHAT
        2 - Join the chat
4. Далее надо ввести логин и пароль. Эта информация будет сохранена в файл
и использована при последующих входах на сайт.
5. Для выхода из чата - ввести сообщение EXIT
